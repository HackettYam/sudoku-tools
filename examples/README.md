# Sudoku Playground Examples

> Fully playable Sudoku games using `@hackettyam/sudoku-tools`
> 
> **IMPORTANT:** These are real, executable game examples — not simple function demonstrations!

---

## 🎮 Available Examples

| Platform | Description | Difficulty | Run |
|----------|-------------|------------|-----|
| **CLI** | Terminal-based interactive game | Medium | `cd cli && pnpm install && pnpm start` |
| **HTML** | Browser-based game (no setup) | Easy/Medium/Hard | Open `html/index.html` in browser |
| **React** | Modern React + TypeScript game | Easy/Medium/Hard | `cd react && pnpm install && pnpm dev` |

---

## 🚀 Quick Start

### Option 1: HTML (Easiest - No Setup)

**Important:** Before opening the HTML example, you must generate the library bundle:

```bash
# From project root, generate the browser bundle
pnpm build:html

# Then open in browser
open examples/html/index.html
# or
firefox examples/html/index.html
```

The HTML example uses the local `bundle.js` file (generated by `build:html`) to run the game with the full library functionality.

---

### Option 2: CLI (Terminal Game)

```bash
cd examples/cli

# Install dependencies
pnpm install

# Start the game
pnpm start
```

**Features:**
- 🎯 Interactive cell selection (row col value)
- 💡 Hint system
- ⏱️ Timer
- 📊 Move counter
- 🎲 Difficulty levels

**Controls:**
```
row col value  - Place number (e.g., "1 2 5")
h, hint        - Get a hint
n, new         - New game
d, difficulty  - Change difficulty
q, quit        - Exit
```

---

### Option 3: React (Modern Web App)

```bash
cd examples/react

# Install dependencies
pnpm install

# Start development server
pnpm dev
```

Opens at `http://localhost:3000`

**Features:**
- 🎨 Modern UI with React 18
- 📱 Responsive design
- 💅 CSS variables for theming
- 🎯 Keyboard navigation (arrow keys + numbers)
- 💡 Hint system (max 5)
- ⏱️ Timer and move counter

## 🐛 Troubleshooting

### CLI: "Cannot find module '@hackettyam/sudoku-tools'"

```bash
# From project root, build the library first
cd ../..
pnpm build

# Then try again
cd examples/cli
pnpm install
pnpm start
```

### HTML: "Cannot read properties of null"

If you see an error about `SudokuTools` being null, you need to regenerate the bundle:

```bash
# From project root
pnpm build:html
```

Then refresh the browser page.

### React: "Module not found"

```bash
# Ensure you're in the correct directory
cd examples/react
pnpm install
pnpm dev
```
