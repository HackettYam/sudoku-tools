## 🖥️ CLI Example (Terminal)

Interactive Sudoku game in the terminal. Demonstrates:
- Puzzle generation with `generatePuzzle()`
- Move validation with `isValidPlacement()`
- Hint system with `getHint()`
- Win detection with `isSolved()`

### Setup and Run

```bash
# Navigate to CLI example
cd examples/cli

# Install dependencies (already done via workspace)
# No additional install needed - uses library via symlink

# Run the game
pnpm start
```

### Game Controls

```
  n              - New game
```


### Key Implementation Details

```typescript
// Import from library
import {
  generatePuzzle,
  isValidPlacement,
  isSolved,
  getHint,
  Difficulty,
  type BoardType,
} from '@hackettyam/sudoku-tools'

// Generate puzzle with solution
const puzzle = generatePuzzle(Difficulty.Normal)
const board = puzzle.board as BoardType
const solved = puzzle.solved as BoardType

// Validate move
const result = isValidPlacement(board, { row, col, value })
if (result.valid) {
  board[row][col] = value
}

// Get hint
const hint = getHint(board, solved)

// Check win
if (isSolved(board, solved)) {
  console.log('🎉 Congratulations!')
}
```

## 🌐 HTML Example (Browser)

Single-file Sudoku game that runs directly in the browser. Demonstrates:
- Local bundle loading of `@hackettyam/sudoku-tools`
- DOM manipulation for board rendering
- Event handling for cell selection and input

### Setup and Run

**Important:** Before opening, generate the library bundle:

```bash
# From project root, generate the browser bundle
pnpm build:html

# Then open in browser
open examples/html/index.html

# Or use a local server
cd examples/html && npx serve .
```

### Features

- **Click** cell to select
- **Type** 1-9 to place number
- **Backspace/Delete** to clear
- **Arrow keys** to navigate
- **H** for hint
- **N** for new game

### Local Bundle Integration

```html
<!-- Load local bundle (generated by pnpm build:html) -->
<script src="bundle.js"></script>
<script>
  // Library available as window.SudokuTools
  const puzzle = SudokuTools.generatePuzzle(SudokuTools.Difficulty.Normal)
</script>
```

> **Note:** Run `pnpm build:html` from project root to generate `bundle.js`. The HTML example will not work without this step.

## 🔧 Common Patterns

All examples follow these patterns when using the library:

### 1. Puzzle Generation

```typescript
import { generatePuzzle, Difficulty } from '@hackettyam/sudoku-tools'

const { board, solved } = generatePuzzle(Difficulty.Normal)
```

### 2. Move Validation

```typescript
import { isValidPlacement } from '@hackettyam/sudoku-tools'

// Object parameter format (v0.1.0+)
const result = isValidPlacement(board, { row: 0, col: 0, value: 5 })
if (result.valid) {
  // Place the value
}
```

### 3. Win Detection

```typescript
import { isSolved } from '@hackettyam/sudoku-tools'

// Requires both current board and solved board
if (isSolved(currentBoard, solvedBoard)) {
  // Puzzle solved!
}
```

### 4. Getting Hints

```typescript
import { getHint } from '@hackettyam/sudoku-tools'

// Requires both current board and solved board
const hint = getHint(board, solved)
if (hint) {
  console.log(`Hint: Place ${hint.value} at row ${hint.row + 1}, col ${hint.col + 1}`)
}
```
